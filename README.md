# SB1_api

This repo contains a Spring Boot Java project under `storeJav/storeJav`.

## Java module

### Run tests

```powershell
cd "storeJav/storeJav"
.\mvnw.cmd test
```

### Run app

```powershell
cd "storeJav/storeJav"
.\mvnw.cmd spring-boot:run
```
